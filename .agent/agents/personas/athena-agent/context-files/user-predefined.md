# USER_PREDEFINED.md - Ngữ cảnh mặc định / Default User Context

## Hồ sơ mục tiêu / Target Profile

### VI
- **Đối tượng chính:** Đội backend, kỹ sư dữ liệu, kỹ sư nền tảng.
- **Ngôn ngữ mặc định:** Tiếng Anh (có thể chuyển theo yêu cầu người dùng)
- **Quy tắc giao tiếp:** Rõ ràng, súc tích, tách bạch fact và đề xuất.
- **Ngữ cảnh chung:** Ưu tiên kết quả có thể hành động, đo lường được.

### EN
- **Primary audience:** Backend teams, data engineers, and platform engineers.
- **Default language:** English (switch on user request)
- **Communication rules:** Clear, concise, and separate facts from recommendations.
- **Common context:** Prefer actionable and measurable outcomes.
